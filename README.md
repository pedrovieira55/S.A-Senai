# S.A-Senai
Meu projeto é fazer um site de academia oline gratuitp para pessoas 
de baixa renda que não consegue pagar uma academia, as funcções são,
o usuario vai poder criar uma conta e fazer uma especie de jogo de perguntas
 e respostas sobre seu físico atualmente e qual fisíco ele almeja chegar
 com isso iremos mandar um cronograma de treino para este usuario para que ele possa chegar perto do corpo que almeja ter.
